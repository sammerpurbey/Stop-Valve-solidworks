# Stop-Valve-solidworks
This project involves the 3D modeling and assembly of a stop valve using SolidWorks. The objective was to develop CAD models of individual components, assemble them into a complete valve, and understand the design and functioning of mechanical assemblies.

Features
- 3D CAD modeling of stop valve components
- Complete assembly of the stop valve
- Engineering drawings of components and assembly
- Standard assembly constraints and part alignment

Components
- Valve Body
- Cover
- Bush
- Rod
- Bridge
- Stud
- Hand Wheel
- V Filler
- Collar
- Seat
- Gland
- Spindle

Software Used
- SolidWorks

Learning Outcomes
- 3D CAD Modeling
- Mechanical Assembly Design
- Engineering Drawings
- Design for Manufacturing (Basic)

Repository Contents
- SolidWorks Part Files (.SLDPRT)
- Assembly File (.SLDASM)
- Engineering Drawings
- Project Report
- Images of the Final Assembly
